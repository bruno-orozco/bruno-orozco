Micro-sofware: creador de polizas

Este micro-sofware, es un creador de pólizas contables. En la empresa se tenía el siguiente problema: se invertían muchas horas de tiempo para realizar calculos básicos. Este programa identifica automáticamente si el monto monetario es un cargo o un abono. Además, toma en cuenta la sucursal y la fecha en la que se realizó la transacción.

Este script fue hecho 100% en Python. Utilicé la librería pandas para la manipulación de datos, y tkinter para obtener una interfaz gráfica amigable para cualquier usuario. Con esto, se consiguió que cualquier persona pueda utilizar el código sin la necesidad de tener grandes conocimientos computacionales. La cabecera prevé los posibles errores que podría cometer el usuario, advirtiendo cómo solucionarlo en caso de cometer cualquier tipo de equivocación.

App web: Gestión de Relación con el Cliente

Esta app, surgió de la ncesidad de recolectar los datos que los vendedores de la empresa generaban. Ya que los empleados pertenecían a diferentes regiones de la república, había un problema de supervisión de la actividad diaria. Este web-site resolvió muchos de estos problemas. Por un lado, el empleado podía ver el avance de sus métricas de productivadad en tiempo real. Con ello, se logró detectar las áreas de oportunidad en el proceso de prospección.

Para la creación de esta app recurrí a HTML, CSS y Javascript para crear el frontend. Para el backend, utilicé el framework Django de Python. Las bases de datos están compiladas con PostreSQL. Además, implementé la automatización de reportes diarios vía email a los líderes de ventas. Aproveché las librerías psycopg2, pandas, y win32com para este propósito.

Cursos de programación: Análisis económico con Python

A medidados de este año, decidí impartir cursos de programación con enfoque económico. Al darme cuenta que no existía un curso de este género, decidí impartirlos. La diferenciación principal respecto a otros, consitió en la identificación de las herramientas necesarias para que un aspirante a economista profesional pueda aprovechar este lenguaje y encontrar un sentido práctico.

En general, reseñaron positivamente mi trabajo. Estudiantes de economía tanto de nivel maestría como de los primeros semestres de economía me agredecieron personalmente por este curso.

Además, compartí en YouTube clases públicas de estadística con Python para apliar esta difusión.
